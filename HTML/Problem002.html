True enough, we can observe that every third Fibonacci number is even.
The sequence of even Fibonacci numbers follows a pattern:
2, 8, 34, 144, 610, ...

Using this pattern, we can generate only even Fibonacci numbers directly.
ie. We can use the sliding window technique to find each new Fibonacci number as 4 times of the second term plus the first term and slide it along.
