Problem Number: 004

Problem Title: Largest Palindrome Product

Problem Definition: A palindromic number reads the same both ways.
The largest palindrome made from the product of two 2-digit numbers is 9009=91x99.

Problem Question: Find the largest palindrome from the product of two 3-digit numbers.



Answer:

Explanation of the algorithm used in pseudocode:

    largestPalindrome = 0

for each number a from 999 down to 100:
    for each number b from a down to 100:  # Avoid duplicate checks
        product = a * b
        if product is a palindrome:
            if product is larger than largestPalindrome:
                largestPalindrome = product  # Update largest palindrome found

return largestPalindrome
